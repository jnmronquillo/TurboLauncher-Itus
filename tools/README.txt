El archivo 29-3f_multiple.ipynb es un archivo creado con Python Notebook.
Para usarlo instalar la distribución de Python Anaconda.
Anaconda ya incluye librerias como NumPy y Pandas
Mas información sobre Pandas:
https://www.youtube.com/watch?v=LBePWE06Wpo&index=39&list=PLea0WJq13cnCno6_21x__20J9kBvrwyog

Copiar 29-3f_multiple.ipynb en el directorio de instalación de Anaconda. Ejemplo;
D:\Anaconda3\Scripts

Dentro de este mismo directorio ejecutar
./jupyter-notebook

El notebook se abrira en un navegador donde se podra crear y ejecutar codigo python.
